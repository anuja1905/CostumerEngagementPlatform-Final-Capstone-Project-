# Customer Engagement & Service Analytics Platform

## Project Overview

The Customer Engagement & Service Analytics Platform is an ASP.NET Core MVC application designed to manage customers and their service tickets efficiently. The application enables organizations to store customer information, create support tickets, track issue status, and improve customer service operations.

## Technologies Used

* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* C#
* Swagger
* JWT Authentication
* HTML
* Bootstrap

## Features

### Customer Management

* Add Customer
* View Customer List
* Edit Customer
* Delete Customer

### Ticket Management

* Create Ticket
* View Ticket List
* Edit Ticket
* Delete Ticket

### Security

* Login Authentication
* Role-Based Authorization
* Admin Dashboard
* User Dashboard
* Access Denied Handling

### API Features

* Customer API
* Ticket API
* JWT Token Generation
* Swagger API Testing

## Architecture

The project follows a 3-Layer Architecture:

1. Presentation Layer (Views)
2. Business Logic Layer (Controllers)
3. Data Access Layer (Entity Framework Core + SQL Server)
